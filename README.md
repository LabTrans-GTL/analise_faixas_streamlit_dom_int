# analise_faixas_streamlit_dom_int
Este é um painel interativo de análise de dados, desenvolvido em Streamlit, para o Plano Aeroviário Nacional (PAN). A ferramenta é projetada para classificar aeroportos brasileiros com base em seu volume de passageiros (embarques e desembarques) e analisar em profundidade as dinâmicas da frota de aeronaves associadas a cada classificação.
